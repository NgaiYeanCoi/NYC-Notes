### 泰勒展开:
$$
 f(x) = f(0) + \displaystyle \frac { f'( 0 ) } { 1! } x +  
 \frac { f''(0) } { 2! } x + 
 \frac { f'''(0) }{ 3! } x + 
 \cdots = 
 \sum_ { n=0 } ^ {\infty }
 \frac { f ^ { (n) } (0) }{ n! }
 x^n
 $$


### $e^x$的求导公式
$$
( e^x ) ' = e^x
$$


 ### 代入$e^x$

 $$
 e^{ x } =
 \sum_ {n=0} ^ { \infty }
 \frac{ x^n }{ n! }
 = 1 + x + \frac{ x^2 }{ 2! } +
 \frac{ x^3 }{ 3! } + \cdots + \frac{ x^n }{ n! }
 $$
